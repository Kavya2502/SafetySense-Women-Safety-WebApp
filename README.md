# SafetySense - Women Safety Web Application

SafetySense is a web-based women safety application designed to assist users during emergency situations through location services, emergency contacts, and safety alerts.

## Features

- Safe Zone Detection using Google Maps
- Navigation and Route Assistance
- Emergency Guardian Contact System
- Safety Alarm with Siren Alert
- Emergency SMS Notification Support
- Responsive and User-Friendly Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- Google Maps API

## Project Structure

- Welcome Page
- Navigation Page
- Safe Zone Page
- Guardians Page
- Safety Alarm Page

## Setup Instructions

1. Clone the repository.
2. Install and run XAMPP or any PHP server.
3. Place the project inside the `htdocs` folder.
4. Configure your Google Maps API Key.
5. Configure your TextLocal API Key in:
   - includes/call1.php
   - includes/call2.php
   - includes/call3.php
6. Start Apache and open the project in your browser.

## Note

This project was developed as an academic project for demonstrating women safety features. SMS functionality and map services require valid API credentials.
