SafetySense - Women Safety Web Application

SafetySense is a web-based women safety application designed to support users during emergency situations. It provides features such as location tracking, safe zone detection, guardian alerts, navigation assistance, and an SOS safety alarm.

The main purpose of this project is to provide quick access to emergency support by helping users identify nearby police stations, navigate safely, and alert trusted contacts when required.

This project was developed using HTML, CSS, JavaScript, PHP, Google Maps API, and location-based services.
